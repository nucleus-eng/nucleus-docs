# House Casing Test

## Should fire (bad casing)

Incubate the starter culture at 260 RPM overnight.
Resuspend the pellet in LB + KAN before induction.
Add Kanamycin from the 1000X stock.
Prepare a 10X dilution of the sample.
Use 2X flasks for the main growth.
Load 6X Laemmli sample buffer.
Soak a 2 kDA dialysis cassette in water.

## Should NOT fire (correct casing)

Incubate the starter culture at 260 rpm overnight.
Resuspend the pellet in LB + Kan before induction.
Add Kanamycin from the 1000x stock.
Prepare a 10x dilution of the sample.
Use a 3 kDa Amicon filter.
See the Slide-A-Lyzer cassette, 3.5K MWCO, for an example.
Add PEG4K to 40% final concentration.
