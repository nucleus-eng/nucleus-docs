# Nucleus Protocols typst template

Branded typst template for lab-ready protocol PDFs generated by
`scripts/build-protocols.py` (see issue #10). Green Nucleus theme, b.next logo,
smallcaps headings, compact table styling.

Vendored from the `nucleus-typst-test` template (b.next / Curvenote, MIT — see
`LICENSE`) and brought in-repo so the pipeline owns and version-pins it rather
than fetching a mutable external `main` branch.

**Pinned dependency:** `@preview/pubmatter:0.2.2`. The original used `0.2.0`,
which panics under typst >=0.8 (`The author or authors must be a array...`)
because its `load` function does string-based `type()` comparisons that newer
typst removed. Do not downgrade.

Rendered via MyST's `exports:` mechanism; the generator points the intermediate
markdown's `template:` field at this directory.
