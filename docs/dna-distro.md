---
title: DNA Distribution
---

The Nucleus DNA Distribution consists of genetic constructs that can be used to engineer synthetic cells. The collection is part of the open source Nucleus synthetic cell package, which also includes protocols and tools for synthetic cell engineering that we validate will work together. 

This DNA distribution has been prepared under OpenMTA with Free Genes and the BioBricks Foundation.

:::{attention}

The documentation for DNA Distribution can be found on the legacy site [here](https://nucleus.bnext.bio/DNA-Distribution-6d3e45ae88b84a038828b815ed54e8bc). 

:::
