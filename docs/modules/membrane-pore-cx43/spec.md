## Overview

The Cx43 membrane pore module produces a self-inserting pore-forming membrane protein that permits the passage of molecules < ~1000 kDa and has the ability to form [gap junctions](https://en.wikipedia.org/wiki/Gap_junction) with neighboring cells.

:::{attention}

This Module has not been validated in Nucleus Cytosol $\ge$ v0.5. Documentation can be found in the following DevNotes:

- [Cx43 Module Validation in Cells](https://doi.org/10.63765/xvxu3274)

:::
